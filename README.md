# بوت المثابر الوزاري (Telegram Bot)

هذا المشروع عبارة عن بوت تيليجرام تعليمي للمراجعة والتحضير للامتحانات الوزارية، مبني باستخدام مكتبة `aiogram`.

## هيكلية المشروع

- `bot.py`: الملف الرئيسي لتشغيل البوت ومعالجة الأوامر.
- `config.py`: يحتوي على إعدادات البوت مثل `BOT_TOKEN`.
- `database.py`: يحتوي على العمليات الخاصة بقاعدة البيانات.
- `questions.py`: يحتوي على بنك الأسئلة والمواد الدراسية.
- `requirements.txt`: قائمة بالمكتبات المطلوبة لتشغيل المشروع.

## كيفية التشغيل

1. قم بتثبيت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```

2. تأكد من وجود ملفات `config.py` و `database.py` و `questions.py` في نفس المجلد.

3. قم بتشغيل البوت:
   ```bash
   python bot.py
   ```

## ملاحظة هامة
يجب التأكد من إدخال `BOT_TOKEN` الصحيح في ملف `config.py` ليعمل البوت بشكل سليم.
